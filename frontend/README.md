# Web Dashboard - PinToWeb

This folder contains a simple HTML frontend for displaying sensor data.

## Function

- Fetches data from backend API at `/api/name/data`
- Displays values in a list (can be extended to graphs)

## How to Run

1. Ensure the backend server is running and accessible
2. Open `index.html` in any browser
3. The page will refresh with new data every few seconds
